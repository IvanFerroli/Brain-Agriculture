    Class PrismaFazendaRepository

    Implementação da interface FazendaRepository utilizando Prisma como mecanismo de persistência.

    Este repositório lida com a criação, atualização, remoção e recuperação de fazendas armazenadas no banco de dados PostgreSQL via Prisma ORM.
