    Class ProdutorService

    Service responsável pelas regras de negócio dos produtores.

    Essa classe consome um repositório que implementa a interface ProdutorRepository, permitindo trocar facilmente a infraestrutura de persistência (in-memory, banco de dados etc.) sem alterar a lógica de negócio.
